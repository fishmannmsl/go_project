`fish_farm`是一个基于`go`语言开发的微服务商城项目,具有`用户服务`、`订单购物车服务`、`商品服务`、`库存服务`、`文件上传服务`。

后端使用`go`语言的`gin`框架进行开发，前端使用`vue2`进行开发

#### 后端

**文件名：fish_farm**

**运行基础配置**

- golang
- docker
  - consul
  - elasticsearch
  - kibana
  - koog
  - jaeger
  - mysql
  - nacos
  - postgresql
  - redis
  - rocketmq
- protoc

项目具体技术栈，请查看`fish_farm`内的`README.md`文件

#### 前端

前端使用 `vue2` 搭建，运行说明在项目文件夹内的 `README.md`

**运行基础配置**

- node
- npm

**文件名：mall-master**

后台管理端
